MATLAB으로 미분방정식풀기

Solve ODE (Ordinary Differential Equation) with MATLAB

MATLAB에는 매우 좋은 기능들이 많습니다. 그 중에 가장 유용하다고 할 수 있는 기능들 중에 하나는 미분방정식을 풀 수 있는 기능입니다. 이 문서에서는 상미분방정식의 해를 구하는 MATLAB 코드에 대해서 다루고 있습니다.

MATLAB has many good functionalities. One of them is a functionality to solve a variety of differential equations. This document is dealing with the MATLAB codes for how to solve ordinary differential equations.

1. dsolve 함수를 사용하는 방법

1. Method to use dsolve function

dsolve라는 함수는 MathWorks 공식 문서에서 찾아보면 'Solve system of differential equations'라고 설명이 나와 있습니다. 따라서 미분방정식 그 자체를 풀어주는 함수이므로 미분방정식을 풀 수 있음을 알 수 있습니다. 해당 함수 사용 방법은 미분방정식을 정의한 후에 그것을 dsolve에 통과시키면 됩니다. 다만, 미분방정식을 정의하기 위해서는 diff라는 미분 함수를 알아야 합니다. 이 함수를 통해서 어떤 변수에 대해서 미분할 것인지, 미분 횟수를 지정할 수 있습니다.

A dsolve function is described as 'Solve system of differential equations' in the MathWorks official document. Therefore, we can know that differential equations could be resolved. After defining an ODE to solve, the solution is found by passing it through the dsolve function. However, to define the ODE, a differentiating function diff is required. Through this function, the derivative variable and the number of difference can be specified. 

clc; clear;
syms y(t) t

Dy  = diff(y, t);    % 1st difference of y with respect to t
D2y = diff(y, t, 2); % 2nd difference of y with respect to t

eqn = D2y + 4*Dy + 3*y == 0; % Define an ODE
% y'' + 4y' + 3y = 0
ySol = dsolve(eqn)

$$$ ySol = C_{2}e^{-t}+C_{1}e^{-3t} $$$

이렇게 풀면 간단하게 풀 수는 있지만, 위의 식을 보면 상수 텀이 남아있습니다. 따라서, 함수를 특정할 수 없기 때문에 그래프를 그릴 수 없습니다. 함수를 특정하기 위해 초기 조건을 지정해주어 미분방정식을 풀 수 있습니다. 초기 조건을 설정하는 방식은 미분방정식을 정의하고 dsolve 함수에 넘겨주는 방식과 유사합니다.

We can solve the ode with this method, but the constant terms remain. Therefoer, we cannot plot a graph because it is impossible to specify the funciton. To specify the function can be reached by declaring some initial conditions. It is similar with the method to define an ode and pass it through the dsolve function. 

clc; clear;
syms y(t) t

Dy  = diff(y, t);    % 1st difference of y with respect to t
D2y = diff(y, t, 2); % 2nd difference of y with respect to t

eqn = Dy/3 == power(t, 2)*y; % Define an ODE
% y'/3 = y*t^2
cond = y(0) == 1;
ySol(t) = dsolve(eqn, cond)
fplot(ySol, [0, 5]) % graph plot

하지만, 초기 조건이 하나뿐이라면 위의 방식으로도 충분히 풀 수 있지만, 초기 조건이 여러 개라면 위의 방식은 부족합니다. 여러 개의 초기 조건을 주기 위해서는 행렬 형식으로 초기 조건을 만들어야 합니다. 그리고 그 초기 조건의 행렬을 초기 조건으로 넘겨주면 됩니다.

If the initial condition is only one, then it is appropriate to solve the given ode with the above method. However, in the case of the multiple initial conditions, it is not proper. To acheive the multiple initial conditions, the condtion of a matrix format is required and we can pass it for the initial condition parameter. 

clc; clear;
syms y(t) t

Dy  = diff(y, t);    % 1st difference of y with respect to t
D2y = diff(y, t, 2); % 2nd difference of y with respect to t

m = 1;  % mass
b = 3;  % damping constant
k = 10; % spring constant

eqn = m*D2y + b*Dy + k*y == sin(t); % Define an ODE
% m*y'' + b*y' + k*y = sin(t)

cond1 = y(0)  == 1;
cond2 = Dy(0) == 0;
cond  = [cond1, cond2];

ySol(t) = dsolve(eqn, cond)
fplot(ySol, [0, 5]) % graph plot

2. 라플라스 변환을 이용하는 방법

2. Method to Use Laplace Transform

라플라스 변환을 통해서 미분방정식을 풀 수도 있습니다. 이는 이론적으로 t-domain과 s-domain을 적절하게 활용할 수 있다는 이점이 있습니다. 라플라스 변환을 위해서는 laplace라는 함수를, 라플라스 역변환을 위해서는 ilaplace라는 함수를 사용할 수 있습니다. 하지만 MATLAB에서는 domain이 항상 t ⇌ s일 필요는 없습니다. 아래 코드의 흐름은 이론적으로 라플라스 변환을 통해서 미분방정식을 푸는 방식과 동일합니다.

Differential Equations can be solved by the laplace transform. This method has a strength to utilize t-domain and s-domain properly in theory. To perform laplace transform, MATLAB has laplace function and ilaplace function. However, the flow of the code below is identical to the method to solve theoritically the differential equations through the laplace transform 

clc; clear;
syms t s Y

m = 1;  % mass
b = 3;  % damping constant
k = 10; % spring constant

cond1 = y(0)  == 1;
cond2 = Dy(0) == 0;

f = sin(t); % external force
F = laplace(f, t, s) % laplace transform 't' to 's'

Y1 = s*Y - cond1;  % laplace transform of y'
Y2 = s*Y1 - cond2; % laplace transform of y''

Sol = solve(m*Y2 + b*Y1 + k*Y - F, Y);
% solve equation m*Y2 + b*Y1 + k*Y - F = 0 with respect to 'Y'
% Result: Y = ~~~
ySol(t) = ilaplace(Sol, s, t); % inverse laplace transform of Sol

fplot(ySol, [0, 5]) % graph plot
grid on
% graph decoration
title("ODE Solution Function", "FontSize", 20)
xlabel("Time")
ylabel("Position")

3. 추가적으로 유용한 함수

3. Other Useful Functions

추가적으로 유용한 함수를 알려드리면서, 마무리하고자 합니다. Unit Step Function (Heaviside Function)과 Dirac-Delta Function (Delta Function)은 MATLAB에 내재되어 있습니다. 하지만, Ramp Function은 MATLAB에 내재되어 있지 않습니다. 이 함수들을 사용하는 방법을 알아봅시다.

Finally, I want to let you know other useful functions for the engineering mathematics. The unit step function (heaviside function) and dirac-delta function (delta function) is built in MATLAB. However, the ramp function is not built-in. Let's figure out how to use these useful functions. 

a = 3;

H = heaviside(t - a); % Unit Step Function
t = 0:0.01:5;
plot(t, H)
%hold on % to plot functions on one window

delta = dirac(t - a); % Dirac-Delta Function
plot(t, delta)
%hold on % to plot functions on one window

ramp = (t-a).*H; % Ramp Function
plot(t, ramp)
%grid on
