Godjunpyo AI Service Documents

[~] : Essential, (~) : Optional

Command Description
/registration Auto registration system to Godjunpyo AI service server.
/check_membership_expiration TA Yoon will check the expected expiration date of your membership.
/ask [question] (web_search) You can ask a question to TA Yoon. Optionally with a website.
/ask_followup [question] You can ask a followup question to TA Yoon considering the previous Q&A.
/ask_concept [question] TA Yoon will describe the concept of the question with a conversation style.
/ask_with_image [question] You must upload your image file and TA Yoon will let you know storing a file with a ðŸ’¾. Then, you can ask a question about the image to TA Yoon. Supported file extension types are .jpg, .jpeg, .png, .gif, .webp, .bmp.
/ask_with_document [question] Document version of /ask_with_image command. Supported file extension types are .pdf, .docx, and .txt.
/search_paper [request] (priority) TA Yoon will search papers related to the given request. The option priority includes 'Latest' or 'Relevance'.
/search_lab [question] TA Yoon will answer a question about given laboratory.
/search_researcher [question] [english_name] TA Yoon will answer questions about the given researcher.
/search_news [request] TA Yoon will search for news relevant to the given request.
/search_english_video [sentence] TA Yoon will search a short English video, which speaks the entered sentence.
/generate_text_to_speech [request] (accent) TA Yoon will pronounce the input texts as following American, British, or Australian accent optionally.
/generate_listening_script [request] (accent) TA Yoon will generate a listening script considering the request as following American, British, or Australian accent optionally.
/generate_image [request] (aspect_rate) TA Yoon will make an image related to your request with the given aspect rate.
/generate_img_to_img [request] (aspect_rate) TA Yoon will make an image of the given aspect rate by referring to the previously uploaded image.
/record_study [study_content] (study_hour) (study_minutes) (feedback) (achievement_rate) Your study will be recorded in the Godjunpyo x AI-Study sheet. The options 'study_hour', 'study_minutes', and 'achievement_rate' allow only integer numbers.
/record_study_alarm [study_content] (study_hour) (study_minutes) Set a timer, alert the end of study session after the time, and record the studying content. You should allow DM from anyone in the server. The options 'study_hour' and 'study_minutes' allow only integer numbers.
/change_nickname Change the nickname saved in the server with the membership code into the current nickname.
/help You can find this page with this command. (same with /help)
Command Description
/search deezer [query] (priority) Search for tracks on Deezer music with priority. Playing YouTube link is prohibitted by a policy of global Discord.
/playlist play [name] Play a saved playlist.
/list Display a list of the current songs in the queue.
/playlist list Display the server's saved playlists.
/playlist list-tracks [name] (page) List the tracks from the specified playlist.
/nowplaying Display the currently playing track.
/skip first Skip one or more tracks in the beginning of the queue.
/stop Stop the player and clear the queue.
/pause on/off on to pause the player and off to resume.
/shuffle on/off on to active shuffle mode and off to deactive shuffle mode.
/shuffle on/off Reshuffle the queue.
/repeat all Repeat all tracks
/repeat single Repeat the same track.
/repeat off Play the queue normally.
/reposition [current-position] [new-position] Change the position of a track in the queue.
/skip first (count) Skip one or more tracks in the beginning of the queue.
