/registration Auto registration system to Godjunpyo AI service server. /registration
/check_membership_expiration TA Yoon will check the expected expiration date of your membership. /check_membership_expiration
/ask [question] (web_search) You can ask a question to TA Yoon. Optionally with a web search. /ask question: Please let me know what important topics are in classical mechanics.
/ask_concept [question] You can ask a concept to TA Yoon. /ask_concept question: What is the concept of Mohr's circle in mechanical engineering?
/ask_followup [question] You can ask a followup question to TA Yoon considering the previous Q&A. /ask_followup question: Then, are the wave equations also important?

/ask_with_document [question] Document version of /ask_with_image command. (With a doc uploaded)
/ask_with_document question: From the document, which one of the people does have the biggest value?
/ask_with_url [question] [url] You can ask a question about a website from the given url. /ask_with_url question: What are the website saying about? url: https://shorturl.at/UgT2X
/search_lab [question] TA Yoon will search laboratories about your question and show them to users. /search_lab question: Please let me know laboratories researching data-mining in the US.

/search_paper [request] (priority) TA Yoon will find out papers related to the given request. Optionally able to set a priority with one of relevance or latest. /search_paper request: Please let me know papers about autonomous driving system and deep learning DNN. piority: Latest
/search_news [request] TA Yoon will find out news articles related to the given request. /search_news request: Please let me know articles about iOS 18.
/search_youtube_godjunpyo [keyword] TA Yoon will find out news articles related to the given request. /search_youtube_godjunpyo keyword: Mohr's Circle
/search_english_video [english_sentence] TA Yoon will find out a video to speak the given english sentence or similar one. /search_english_video english_sentence: This is harassment
/generate_text_to_speech [english_text] (accent) TA Yoon will generate the pronunciation of the given English text. Optionally able to set the pronunciation accent with one of American, British, and Australian. /generate_text_to_speech english_text: deduce the annals from 1620 accent: American
/generate_listening_script [request] (accent) TA Yoon will generate the listening script related to the given request. Optionally able to set the pronunciation accent with one of American, British, and Australian. /generate_listening_script request: Possible listening script on the TOEFL listening section. It is related to the topic of geology and is 3 minutes long. accent: Australian
/record_study [study_content] (study_hours) (study_minutes) (feedback) (achievement_rate) Record study content at the recording spread sheet. Optionally able to write study hours, study minutes, feedback, and achievement rate. /record_study study_content: Multiplication of matrices in Linear Algebra study_hours: 1 feedback: Computation was slightly complicated, but I've totally understood it. achievement_rate: 90
/record_study_alarm [study_content] (study_hours) (study_minutes) TA Yoon will alert the study completion to you after the set time. Optionally able to set study hours, and study minutes, but at least one should be filled. /record_study_alarm study_content: PDE of 2-dimensional circular membrane in Engineering Mathematics study_hours: 1 study_minutes: 30
/generate_image [request] (aspect_ratio) TA Yoon will make an image related to your request with the given aspect rate. /generate_image request: internship aspect_ratio: Landscape
/generate_img_to_img [request] (aspect_ratio) TA Yoon will make an image of the given aspect rate by referring to the previously uploaded image. (With an img uploaded)
/generate_img_to_img request: students having a discussion are sitting like the given image aspect_ratio: Portrait

Command Description Example
t! top (page_num) Show the ranking, which is corresponding to the page number, of the Godjunpyo AI server. At most 10 users could be shown per page. The default value is 1. t! top 1
t! profile (user) Show a global profile of the user. Tatsu bot is a global-wide bot. So this profile does not reflect the only Godjunpyo AI discord server. User should be a mentioned form (@) or universal id (unique one). The default value is the user himself. t! profile mutexlock_
t! rank (user) Show a server scale profile of the user. User should be a mentioned form or universal id (unique one). The default value is the user himself. t! profile @MutexLock-G1752
t! wallet (user) Display wallet, for tatsu bot, including credits, tokens, and server points. These kinds of currency would be utilized to buy a badge later. The default value is the user himself. t! wallet
t! reputation [user] Vote up the reputation of the user. If you are given help from another person, we recommend giving a reputation to that one. User should be provided and cannot be himself. You can give a reputation to a person every 12 hours. To use this command, you must login to the dashboard of this site at least once. t! reputation mutexlock_

;compile [lang]
```(lang_type)
[code]
